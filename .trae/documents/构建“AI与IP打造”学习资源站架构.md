# 个人 AI 内容创作学习中心构建计划

根据您提供的详细架构调整，我们将构建一个极简、清新且充满个人特色的学习资源站。

## 1. 技术栈确认

* **核心框架**: Next.js 14 (App Router)

* **样式系统**: Tailwind CSS (极简/清新风格)

* **内容引擎**: MDX (统一管理资源与博客)

* **图标库**: Lucide React

## 2. 页面与功能实现规划

### 2.1 首页 (Home)

* **介绍区**: 核心主张

* **资源分类导航**: 清晰展示 "AI 技术学习" 与 "内容创作/IP" 入口。

* **精选与路径**: 展示 "个人推荐" 和 "学习路径" 组件。

* **最新动态**: 自动抓取最近更新的资源或博客。

### 2.2 学习资源库 (Library)

* **分类体系**: 实现 AI (基础/工具/进阶) 与 内容 (技巧/运营/变现) 的多级分类。

* **筛选功能**: 开发基于 Tags 的筛选器 (按主题、难度、形式)。

* **资源详情**: 支持 MDX 渲染，展示工具指南、模板下载和实战案例。

### 2.3 个人博客 (Blog)

* **内容板块**: 区分 "学习笔记"、"往期文章" 和 "成长历程"。

* **阅读体验**: 专注于文字阅读的极简排版。

### 2.4 关于我 (About)

* **静态页面**: 展示个人背景、创作理念、联系方式及建站初衷。

## 3. 开发步骤

1. **初始化**: 搭建项目骨架，配置 Tailwind 极简主题。
2. **组件开发**: 优先开发导航、个人 Hero、资源卡片。
3. **内容层接入**: 设置 MDX 文件读取逻辑，建立 `content/resources` 和 `content/blog` 目录。
4. **页面组装**: 按照首页 -> 资源库 -> 博客 -> 关于的顺序完成页面。
